What Models Bring, Where They Fail

Strengths
Vast training, in-context learning
Reasoning, analysis, code generation
Speed, scale, pattern recognition
Failure modes
Knowledge cutoff, stale facts
Hallucination, confident wrong answers
Non-deterministic output

The 4D Framework · Dakan & Feller

The 4D Cycle

Delegation

What's worth doing with AI, and how do we split the work?

Description

Communicate the vision, requirements, constraints, and how the AI should behave.

Discernment

Critically evaluate every output. Is it useful, correct, right approach?

Diligence

Take responsibility for the final product. Accuracy, honesty, ethical use.

Delegation → Description → Discernment → Diligence → (repeat)

Why This Matters

Please read the code.

Dex Horthy, on discernment

2026 is the year of no more slop. We have a profession to uphold.

On diligence

Agency raises the stakes. You are vouching for code you did not type. Discernment and diligence are the load-bearing skills.

Part 2

Claude Code

An agentic coding tool that reads your codebase, edits files, runs commands, and integrates with your dev tools. Terminal, IDE, desktop, web, mobile.

What It Does

Build & fix

Describe what you want. It plans, edits across files, runs tests, iterates.

Git-native

Stages changes, writes commit messages, opens pull requests.

Connect anything with MCP

Design docs, tickets, Slack, custom internal tools.

Customise with CLAUDE.md

Coding standards, architecture notes, libraries, review checklists.

Agent teams

Spawn sub-agents that work different parts of a task in parallel.

Pipe & script

Composable CLI. Pipe logs in, run in CI, chain with other tools.

Available Everywhere

Same engine, same settings, same CLAUDE.md across every surface.

Terminal · primary VS Code JetBrains Desktop app Web iOS
1 account / 1 config / N surfaces

Install & Run

# macOS, Linux, WSL
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell
irm https://claude.ai/install.ps1 | iex

# Homebrew
brew install --cask claude-code

# Run it
cd your-project && claude

Log in on first use. Docs: code.claude.com/docs
