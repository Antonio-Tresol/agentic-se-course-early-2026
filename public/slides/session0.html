A fast recap of foundational concepts: what fluent AI use looks like and the 4D framework for reasoning about how we work with AI.

Fluent AI Use Is...

Effective

Achieves the desired outcome.

Efficient

Three Ways to Interact with AI

01 Automation. AI does work for you following explicit instructions.
02 Augmentation. You and the AI collaboratively work together.
03 Agency. AI works independently on your behalf.

This course focuses on the agency end of the spectrum: coding agents.

What Makes Generative AI Powerful

Anthropic Goals of Fine-Tuning

Three principles guide how Claude is trained to behave.

H Helpful. Useful, accurate, and responsive to what you need.
H Honest. Truthful, transparent about uncertainty and limitations.
H Harmless. Avoids causing harm, respects safety boundaries.

The AI Context Window

Everything the model can see in a single interaction.

Your prompts. The instructions and questions you provide.
AI responses. Everything the model has said so far in this conversation.
Any other info you've shared. Files, code, images, tool results.

Context is finite. Managing it well is a core skill you will develop throughout this course.

The 4D Framework

Four Core Competencies for AI Fluency

Developed by Rick Dakan (Ringling College) and Joseph Feller (Cork University Business School). A cycle of interconnected skills for effective human-AI interaction.

The 4D Cycle

Delegation

Deciding what to do with AI and how to divide the work.

Description

Communicating your vision, requirements, and constraints to the AI.

Discernment

Critically evaluating AI output for quality, relevance, and correctness.

Diligence

Taking responsibility, ensuring accuracy, honesty, and ethical use.

4D Framework

Delegation

So... who does what?

Understanding the problem you are trying to solve
Understanding the capabilities of available AI tools
Breaking down complex work into smaller parts
Making strategic decisions about who does what
Choosing the right mode of interaction
4D Framework

Description

Effectively communicating your vision to AI systems.

What you want the final output to be

Clear prompts defining desired outputs with specific characteristics.

How you want the AI to approach the task

Iterative, dialogic prompting for multi-step collaborative work.

How you want the AI to behave

Defining AI behaviors, constraints, and guidelines for the interaction.

4D Framework

Discernment

Critically evaluating AI-generated outputs.

Is the output useful and correct?
Is the AI taking the right approach?
Is the AI behaving as desired?

This becomes critical with coding agents. You must be able to evaluate code you did not write. As Dex Horthy says: "Please read the code."

4D Framework

Diligence

Taking responsibility and vouching for the final product.

Ensuring accuracy and taking responsibility
Honesty and transparency about AI involvement
Ethical use and critical awareness

Part 2

Setup Claude Code

Claude Code is an agentic coding tool that reads your codebase, edits files, runs commands, and integrates with your development tools. Available in your terminal, IDE, desktop app, and browser.

What Can Claude Code Do?

Build features and fix bugs

Describe what you want. Claude plans the approach and writes code across multiple files.

Commits and pull requests

Works directly with git. Stages changes, writes commit messages, creates PRs.

Connect tools with MCP

Read design docs, update tickets, pull data from Slack, or use custom tooling.

Customize with CLAUDE.md

Set coding standards, architecture decisions, preferred libraries, review checklists.

Agent teams

Spawn multiple agents that work on different parts of a task simultaneously.

Pipe, script, automate

Composable CLI. Pipe logs into it, run it in CI, chain it with other tools.

Available Everywhere

Same engine, same settings, same CLAUDE.md across all surfaces.

Terminal

Full-featured CLI. Our primary tool for this course.

VS Code

Inline diffs, @-mentions, plan review.

JetBrains

IntelliJ, PyCharm, WebStorm and more.

Desktop App

Standalone app, visual diffs, multiple sessions.

Web

No local setup. Run tasks from anywhere.

Mobile

Continue sessions from your phone via iOS app.

Let's Install It

You need a Claude subscription or Anthropic Console account.

# macOS, Linux, WSL (recommended)
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell
irm https://claude.ai/install.ps1 | iex

# Or via Homebrew
brew install --cask claude-code

# Then start it in any project
cd your-project
claude

You'll be prompted to log in on first use. Full docs at code.claude.com/docs/en/overview
